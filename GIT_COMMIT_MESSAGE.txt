## Database Changes

### Tables Modified
1. **teachers** table
   - ✨ Added: `is_adviser` (boolean, default false)
   - ✨ Added: `advisee_count` (int, default 0)  
   - ✨ Added: `assigned_since` (date, nullable)
   - 🗑️ Removed: `teacher_type` enum (was: 'Ordinary', 'Adviser', 'Both')
   - ✅ Added index: `KEY idx_is_adviser (is_adviser)`

2. **students** table
   - No changes (preserved from previous CSG merge)

3. **users** table
   - No changes (unchanged)

### Sample Data Updates
- 3 advisers: `is_adviser = true`, `assigned_since = 2020-06-01 or 2021-06-01`
- 3 ordinary teachers: `is_adviser = false`, `assigned_since = NULL`
- All teachers: `advisee_count` initialized appropriately

## Form Fields by Role

### Student (role_id = 4)
**Base:**
- name, email, password, phone, institute_id

**Specific:**
- student_id (required, unique)
- year_level (required, select)
- course (required, text)
- adviser_id (optional, select from advisers)
- enrollment_date (optional, date)
- expected_graduation (optional, date)

### Teacher (role_id = 5)
**Base:**
- name, email, password, phone, institute_id

**Specific:**
- employee_id (required, unique)
- department (required, text)
- specialization (optional, text)
- office_location (optional, text)
- office_phone (optional, tel)

### Adviser (role_id = 2 - Admin/Adviser)
**Base:**
- name, email, password, phone, institute_id

**Specific:**
- employee_id (required, unique)
- department (required, text)
- specialization (optional, text)
- office_location (optional, text)
- office_phone (optional, tel)
- assigned_since (optional, date)

### CSG & Super Admin (role_id = 1, 3)
**Base only** (no role-specific fields)

## Migration Guide for Existing Data

### If Existing Teachers Table Has Data
```sql
-- Backup first
CREATE TABLE teachers_backup AS SELECT * FROM teachers;

-- Update is_adviser from teacher_type
UPDATE teachers SET is_adviser = 1 WHERE teacher_type IN ('Adviser', 'Both');
UPDATE teachers SET is_adviser = 0 WHERE teacher_type = 'Ordinary';
UPDATE teachers SET assigned_since = CURDATE() WHERE is_adviser = 1 AND assigned_since IS NULL;

-- Verify
SELECT * FROM teachers;

-- Drop old column (if no longer needed)
-- ALTER TABLE teachers DROP COLUMN teacher_type;
```
